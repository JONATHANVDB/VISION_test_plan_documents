1. Description  
The Magics’ Vision Series portfolio contains products 
which are to be used as core building blocks of a 
radiation -hardened camera. Next to the standalone ICs 
in the Vision Series, Magics also offers full -blown 
electronic Vision modules, containing all ICs from the 
Vision Series, integrated in the form of a three -PCB 
stack up. The modules are the electronic heart of a 
radiation -hardened camera and contain at minimum 
the following Magics’ ICs from the Vision Series:  
• MAG -IMG002X1 -NC: Full HD CMOS rad -hard 
image sensor  
• MAG -PSU00001 -NP: rad -hard DC/DC 
converter controller  
• MAG -CXP00002 -NP: CoaXPress serializer IC  
Different variants with different features are offered as 
an end product (overview of product variants can be 
found in section 7), of which some also contain ICs from 
the Magics’ Motion Series. By incorporating the Motion 
Series ICs, additional functionality, such as motorized 
lens control, can be added to the module.  
Next to the Magics’ ICs all the supporting electronics, as 
described in the application diagrams in the respective 
IC datasheets, are integrated into the module. These 
components include passives, such as resistors, inductors and capacitors, as well as a XTAL oscillator for 
clock reference input. In addition, there is a connector 
to supply power (9V) and a HD -BNC connector to 
interface a 75 -Ohm coaxial cable for video transmission 
(downlink) and configuration data (up link).  
2. Key features  
• Modular design consisting of three PCBs 
containing Magics’ ICs and supporting 
components.  
o Electronic module can be integrated as is in 
a rad -hard camera. No additional electronic 
components required!  
• Full HD (1920x1080) CMOS image sensor  
o >2Mpixel  
o Up to 30 FPS in Full HD mode  
o Electronic rolling shutter  
o > 60dB dynamic range  
o Available in monochrome and color version  
▪ RGGB Bayer pattern for color version  
• Features  
o Image flipping  
o Dark pixel readout option  
o Programmable integration time  
o Configurable window modes (FHD, HD, VGA , 
etc.)  
• Configurable operating mode  
o Triggered, burst and continuous  mode  
• CoaXPress video interface: 1.25Gbps single line 
CXP 2.0 device side  interface  
o 1.25Gbps downlink communication  
o 20Mbps uplink communication  
o 75-Ohm coax cable – up to >150m depending 
on the quality of the cable  
o GenICam compliant  
• On-board X -TAL clock of 20 MHz  
• Single supply operation : 9V input voltage  
o Internally 3.3V and 1.8V supply voltages are 
generated  
• Operating temperature: 0 to 50 degC ambient  
• Different flavors with different functions 
available.  
o E.g. integrated motorized lens control.  
• Radiation hardness:  
o > 1MGy TID (Si)  
o Single -event tolerance by design

3. MAG -VIS modules and block diagrams  
The core products of Magics Technologies are ICs and they form the base for all of our derived products. The ICs in the 
Vision Series are already listed above and are depicted in Figure 2. 
 
Figure 2: Three main ICs as part of the Magics' Vision Series. Note: MAG -IMG001X1 -NC (JLCC -68 package) has become obsolete.  
3.1 MAG -VIS modules overview  
Magics offers MAG -VIS modules in the form of a three -PCB stack, of which each PCB has his own function and 
components. Below the three PCBs in the stack of the MAG -VIS modules are listed:  
- PCB1 : this PCB contains the MAG -IMG002x1 -NC FULL HD image sensor  
- PCB2 : this PCB contains a MAG -PSU00001 -NP controller to generate the 1.8V supply for the imager (PCB1) and 
optionally additional functions are added to this PCB as well.  
o Different versions available, for example with a MAG -DRV00002 -NP added to the design. See Table 2. 
- PCB3 : this PCB contains the MAG -CXP00002 -NP CoaXPress -compatible serializer chip, and a MAG -PSU00001 -
NP to generate the 3.3V supply to supply the imager and the serializer.  
To date, Magics offers three different variants of the MAG -VIS modules  (six if differentiated between mono and color) , 
and these modules are depicted in Figure 3. 
 
 
Figure 3: MAG -VIS modules - three different variants.
