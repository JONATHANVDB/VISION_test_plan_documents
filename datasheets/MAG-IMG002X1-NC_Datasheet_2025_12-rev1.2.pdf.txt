3. Description  
The MAG -IMG00 2X1-NC is a radiation -hardened F ull 
HD CMOS digital image sensor  with an active -pixel array 
of 1920Hx1080V . It can come in both a monochrome  
(MAG -IMG00 241-NC) as in a color version (MAG -
IMG00 251-NC). Both versions implement rolling 
shutter readout strategy . 
The image sensor outputs raw sensor image data 
through a parallel CMOS output interface (3v3) , which 
is configurable to output 10 or 12 bit pixel  data. 
Synchronization signals such as the pixel clock (P IXCLK), 
data valid (DVAL), line valid (LVAL) and frame valid 
(FVAL) are also sent  along through the same output 
interface , which can run up to 100 MHz . The imager can 
be configured to capture data in trigger mode, in which 
it can capture one image or multiple images (burst 
mode), or in continuous mode (video mode). It also 
includes features such as image mirroring/reversing  
and sub -sampling /decimation  to ease the integration 
for the end application . 
All the above configurations can be programmed 
through a SPI interface which can run up to 20 MHz.

5. Block diagram  
The block diagram of the imager’s internal s is depicted in Figure 3. The core of the device is the pixel array, together 
with the peripherals to drive and read the pixels.  The latter circuitry includes the column load, sample and hold and the 
column ADC. SRAM is implemented to store the data to be sent out through the parallel output bus as raw image data.  
 
Figure 3: Imager block diagram  
 
The signals that are required to select the right column and row to be read , are generated by the sequencer in the 
digital. This block also generates the signals that are used to drive the reset and selection signals of the pixels , and the 
signals that are use d for the sample and hold and the ADC. The sequencer defines  the biggest part of the digital circuitry 
on chip. Next to the sequencer, the digital contains an SPI interface together with a register map which is accessible 
through this SPI interface. It allows the user to configure the device in the operating mode wanted , inclu ding the 
implemented features . The digital  operates at the digital supply voltage VDDD of 1.8V.  
Next to the digital, the imager contains analog circuitry to generate on -chip reference currents and voltages, and 
reference clocks as well. The chip contains a PLL which generates the clock for the digital, as well as the clocks required 
to drive the ADCs , the output MUX and the output interface. The PLL requires an external clock reference of 20 Mhz.
